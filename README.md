# Ruth 路思

让智慧不再遥远，持续滋养你。

## 简介

Ruth 是一个让智者触手可及的对话应用。第一位深度建设的智者是乔治·凯利（George Kelly）——一位被严重低估的心理学家，个人建构心理学的创始人。

### 核心理念

- **问智者，读智者**：与历史上的智者对话，理解他们的思想框架
- **见天地，见自己**：通过智者的眼睛看世界，通过构念系统看自己
- **持续滋养**：不是一次性的答案，而是持续的思考陪伴

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无需构建步骤
- 响应式设计，适配移动端

## 功能

- 智者介绍页
- 沉浸式对话界面
- 核心概念解析
- 更新日志
- 用户反馈系统
