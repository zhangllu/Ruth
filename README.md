# Ruth 路思

让智慧不再遥远，持续滋养你。

## 简介

Ruth 是一个让智者触手可及的对话应用。第一位深度建设的智者是乔治·凯利（George Kelly）——一位被严重低估的心理学家，个人建构心理学的创始人。

## 功能

- ✅ 与乔治·凯利真实对话
- ✅ 基于个人建构心理学的智能回应
- ✅ 对话历史记录
- ✅ 响应式设计，适配移动端

## 技术栈

- **前端**: 纯 HTML + CSS + JavaScript
- **后端**: Vercel Edge Functions
- **AI**: 智谱 AI GLM-4 模型
- **部署**: Vercel

## 本地开发

```bash
# 安装依赖
bun install

# 启动开发服务器
bun run dev

# 访问 http://localhost:3000
```

## 环境变量

需要配置智谱 API Key：

```bash
# .env
ZHIPU_API_KEY=your_api_key_here
```

## 部署

```bash
# 部署到 Vercel
bun run deploy
```
