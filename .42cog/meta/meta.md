# 项目元信息 (Meta)

## 基本信息

**项目名称**: 问凯利 (Ask Kelly)

**项目标语**: 和凯利对话，看见自己建构的世界

**版本**: 2.0

**创建日期**: 2025-02-01

**负责人**: 随机漫步

## 项目描述

问凯利是一个以心理学家乔治·凯利（George Kelly）为核心的智慧对话与知识网站。用户可以与基于个人建构心理学（Personal Construct Psychology）训练的 AI 凯利直接对话，了解这位被严重低估的心理学家及其理论，并阅读作者关于凯利思想的系列文章。项目旨在让智者的思考方式触手可及，持续滋养用户的生活与工作。

## 核心模块

1. **凯利个人介绍** — 乔治·凯利的生平、学术背景与贡献
2. **个人建构心理学** — 核心理论介绍，包括建构替代论、建构体系、CPC 循环等
3. **凯利文章** — 作者随机漫步撰写的关于凯利思想的系列文章与学习笔记
4. **AI 对话** — 基于智谱 AI（GLM-5.1）构建的凯利人格对话接口，用户可以直接向凯利提问

## 目标用户

- 对心理学感兴趣的普通读者，希望了解被低估的心理学理论
- 教育工作者和教学设计者，关注建构主义学习理论
- 个人成长实践者，希望获取新的思维框架
- 凯利个人建构心理学的研究者和爱好者

## 核心价值主张

1. **让智者触手可及** — 不仅学习凯利的理论，更通过 AI 对话与凯利"面对面"交流
2. **深度理论内容** — 系统化的个人建构心理学知识库，持续更新
3. **知行合一** — 将凯利的建构思想融入日常生活与工作决策

- 框架: 纯 HTML + CSS + JavaScript（v1/v2 版本）

**后端**:
